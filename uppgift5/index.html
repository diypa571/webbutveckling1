Cascading Style Sheets (CSS)

CSS (Cascading Style Sheets), det ä ett stilmallspråk, men det uppfattas inte som ett riktigt programmeringsspråk som java. CSS används tillsammans med olika HTMl-elementen.

Vi använder CSS för att kunna styra olika HTML-element och vi kan skapa snygga repsonsiva webbsidor med den.
Det finns två metoder för att ha CSS kod, vi kan skriva (inline) CSS, att skriva koden direkt i html-documenten för de olika HTML-element eller vi kan också använd CSS stilmallar, alltså att CSS som externa filer.
Så här ser ut när vi skriver CSS som inline.

Exempel:
<h1 style="background-color:green;"> HELLO WORLD </h1>
Då får elementet <h1> en grön backgrundsfärg.
Själv tycker jag att man skan använda CSS filer som bara externa filer. Då inkluderas den externa CSS filen med link.
Följande exempel. <link href="css/style.css" rel="stylesheet">
Relevant att ha länken mellan <head> och </head>
Det finns många fördelar med att göra de, nedan kommer jag att skriva några.

Följande fördelar med att använda CSS som externa filer.

1- Det blir lättar att skriva koden.
2- Enklare att få översikt på alla olika HTML-element.
3- Att fixa error blir också mycket lättare. Blir det fel på CSS koden, ja, man vet då att CSS koden finns i style.css
4- Externa filerna behöver inte vara på samma webserver.
