KORT OM CSS

CSS (Cascading Style Sheets), det ä ett stilmallspråk, men det uppfattas inte som ett riktigt programmeringsspråk som java. CSS används tillsammans med olika HTMl-elementen och för att kunna styra till exempel hur olika elementen ska få se ut.

Varför finns CSS?

CSS finns i webbläsarens renderingsmotorer (rendering engines). Olika webbläsaren kan ha support för olika versioner av CSS. Jag kommer ihåg att gamla versioner av internet explorer hade inte support för senaste versionen av CSS. Det var ett problem för Microsoft, folk började använda webbläsarna Firefox och Chrome istället. Webbläsaren Edge har support för CSS3 in renderingsmotorn, vilket är bra. Renderingsmotorer är relevanta, för att alla HTML-element kommer att tolkas, representeras som object. Alltså Document Object Model, förkortas som DOM. För oss programmerat javascript, att ha olika HTML-element representerat som objekt, är bra viktigt för att då javascript kommer att kunna hantera HTML-elementen. För CSS, tanken är ungefär samma, CSS Object Model, förkortas som CSSOM.

Hur använder vi oss av CSS??

Vi kan använda CSS. CSS kan används direkt eller som externa CSS länkar i html filer. Normalt ska CSS koden finnas mellan <head> Här ska CSS skrivas. </head> Vi kan skapa CSS regler som kan bestå av en klass eller ett id, med två st lockiga parenteser som får innehålla CSS egenskaper och värden. .webbutveckling1 {}, Vi kan skriva olika CSS klasser. #webbutveckling1{} , är skapa en id=”webbutveckling1”, skrivas med en fyrkantstagg (hastag) och det är ett id som kommer att skapas i CSS koden. När vi använder oss av HTML-elementen direkt, det här metoden kallas för CSS selektorer. Exempel. p {background-color:green} Här väljer vi p, alla p elementen i html-dokumentet får ha en grön bakgrundsfärg. Background-color, är en atribut. Green, är ett värde till background-color.

Vilka versioner finns det?

CSS1, har funnit sedan 1996, man kunde använda den för att kunna till exempel ha olika färger på texter. CSS2, med den här versionen blev det möjlig att ha olika positioner för olika element. CSS3, den utvecklades år 1999 och med den här versionen blev det möjlig att ha olika positioner för olika element.

Varför finns det olika versioner? (Varför är det inte bara en?)

Varje ny version innehåller nya funktioner samt existerande funktioner kan ha blivit bättre. Tekniken går framåt, olika mobila enheter produceras, webbsidorna får anpassa sig efter olika enheterna.

Varför är det viktigt att veta vilken version som en viss CSS-egenskap tillhör? (Kan detta påverka webbplatsen?)

Det kan vara bra att veta, moderna webbläsaren har support för CSS3. Gamla webbläsaren till exempel,internet explorer kommer inte att ha support för Grid. Webbsidor som innehåller CSS grid funktioner kommer att se konstiga. Till exempel, med version CSS3, vi kan ha runda hör på en div box, funktionen existerar inte i äldre versionen. Den nya versionen innehåller grid vilket bra för att kunna skapa olika responsiva stilmallar som anpassar sig efter olika enheters storlek på bredden. Man brukar rekommendera att använda CSS3, för att som skrev ovanför, moderna enheter har support för CSS3.
